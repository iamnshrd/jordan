# Library

This library holds local source material for the Jordan Peterson–style agent.

## Folders
- `books/` — full books or long-form texts
- `articles/` — essays, interviews, articles
- `excerpts/` — selected passages, quotes, clipped sections
- `notes/` — agent-authored or user-authored reading notes and synthesis
- `incoming/` — raw newly added files before cleanup/normalization

## Suggested workflow
1. Drop new files into `incoming/`
2. Rename/normalize them
3. Move them into the correct folder
4. Optionally create notes/excerpts for high-value sections
5. Later, add indexing/retrieval if the library gets large

## Supported idea
Keep personality in prompt files, but keep actual content in the library.
